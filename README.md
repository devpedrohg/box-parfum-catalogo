# BOX PARFUM

Catálogo premium responsivo desenvolvido para a BOX PARFUM.

## Sobre o projeto

Sistema desenvolvido manualmente em HTML, CSS e JavaScript com foco em experiência premium mobile e finalização rápida via WhatsApp.

* catálogo responsivo
* carrinho inteligente
* checkout personalizado
* finalização automática via WhatsApp
* pesquisa de produtos
* filtros e ordenação
* experiência mobile premium

* JavaScript
